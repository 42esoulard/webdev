It is an OFFSET (it moves the element away from the other element) (or top/left/right). The document's normal flow is maintained and other elements will be moved as well. p { position: relative; bottom: 10px; } Absolute: Locks the elements to its closest positioned ancestor (parent must have position: relative;). Will be ignored by other elems, element is removed from the normal flow #searchbar { position: relative; top: 10px; right: 10px; } Fixed: like absolute, but element doesn't even move when scrolling #navbar { position: fixed; top: 10px; left: 10px; } Float: left or right. Pushed element to that side. #right { float: right; width: 50%; } Order overlapping elements: z-index #bla { z-index: 2; } #blou { z-index: 1; } Center element: margin: auto; //works for images too, by adding display: block; so they're treated as a block --------> COLOR hsl(degree, saturation, lightness) (degree is on the color wheel, ex 170) background-color: hsl(180, 90%, 35%); Gradient: (direction in degrees, ex 68deg) background: linear-gradient(gradient_direction, color 1, color 2, color 3, ...); Repeating gradient (ex: stripes) repeating-linear-gradient( 90deg, //direction yellow 0px, blue 40px, green 40px, red 80px ); Set an image as background: background: url(https://cdn-media-1.freecodecamp.org/imgr/MJAkxbh.png); --------> COLOR THEORY: Complementary colours pop. The split-complementary color scheme: starts with a base color, then pairs it with the two colors that are adjacent to its complement. The three colors provide strong visual contrast in a design, but are more subtle than using two complementary colors. !Careful! have high enough contrast for accessibility. Careful with colours hard to distinguished for colorblinds (green vs yellow/brown vs yellow) --------> SHAPES: css can be used to make shapes, by modifying a box's appearance. ::before and ::after are pseudo elements which add an element to an element. ex: heart { blabla: shape; } heart::before { blabla: rectangle shape; } --------> ANIMATIONS: Keyframes: #anim { animation-name: colorful; animation-duration: 3s; animation-fill-mode: forwards; //allows anim not to stop animation-iteration-count: 3; //nb of iterations.
